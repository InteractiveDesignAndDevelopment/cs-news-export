# cs-news-export

This project exports the CS News Articles for importing into the new WordPress News site.

## Sample cs-authors-export.xml file.

```xml
&lt;?xml version="1.0"?&gt;
&lt;authors&gt;
  &lt;author&gt;
    &lt;first_name /&gt;
    &lt;last_name /&gt;
    &lt;role /&gt;
    &lt;login /&gt;
    &lt;email /&gt;
    &lt;display_name /&gt;
  &lt;/author&gt;
&lt;/authors&gt;
```

## Sample cs-articles-export.xml file.

```xml
&lt;?xml version="1.0"?&gt;
&lt;articles&gt;
  &lt;article&gt;
    &lt;title /&gt;
    &lt;content /&gt;
    &lt;excerpt /&gt;
    &lt;images /&gt;
    &lt;pageid /&gt;
    &lt;categories /&gt;
    &lt;tags /&gt;
    &lt;post_date /&gt;
    &lt;post_slug /&gt;
    &lt;post_author /&gt;
  &lt;/article&gt;
&lt;/articles&gt;
```
